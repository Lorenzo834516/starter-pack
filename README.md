# TS-Backend-Starter

Este es un *Starter Pack* profesional para crear aplicaciones Backend utilizando *Node.js* y *TypeScript*. Está configurado para ahorrar tiempo en el setup inicial y seguir las mejores prácticas.

##  Tecnologías utilizadas
* [Node.js](https://nodejs.org/)
* [TypeScript](https://www.typescriptlang.org/)
* [Dotenv](https://www.npmjs.com/package/dotenv) (Manejo de variables de entorno)
* [Nodemon](https://nodemon.io/) (Reinicio automático del servidor)
